Instructions

Fill out the list with the fish you've caught by clicking the checkboxes next to them.

The box underlined in blue will check every size for every mutation for that fish, excluding the Midas variant.

The box underlined in red will check every size for that specific mutation.
When you have filled out all the fish you've caught for a location, make sure to click the button.
This will update the bar graph for that location. Make sure you do this any time you are done adding fish!

Data
